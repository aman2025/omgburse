
# 前端服务启动步骤

## Nodejs版本要求
```
Node v8.11.1 以上
```


## 全局安装vue-cli 
```
npm install -g @vue/cli
```

安装后查看版本
```
vue -v
```
目前是@vue/cli 4.5.9


## 在项目目录下执行命令
```
npm install
```

### 启动服务
```
npm run serve
```
首页
http://localhost:8003  

登录
http://localhost:8003/login

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
